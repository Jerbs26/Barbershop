# Urban Razor Barbershop System

## Project Description

Urban Razor Barbershop System is a mobile-based application designed to improve barbershop operations and customer experience. The system allows users to book appointments, order products, receive hairstyle recommendations using Camera Finder, and provide ratings and feedback. It also includes an administrator panel for managing bookings, services, products, barbers, schedules, and customer feedback.

## Objectives

* Provide an easy and convenient appointment booking system
* Allow customers to purchase barbershop products online
* Recommend suitable hairstyles using Camera Finder technology
* Collect customer ratings and feedback for service improvement
* Help administrators manage daily barbershop operations efficiently

## User Features

* User Registration with Email OTP Verification
* Login and Forgot Password
* Dashboard Access
* Appointment Booking with GCash Payment
* Product Ordering with Checkout
* Booking and Order History
* Ratings and Feedback System
* Camera Finder for Hairstyle Recommendations

## Admin Features

* Admin Login
* Dashboard and Analytics
* Booking History Management
* Barbers Management
* Services Management
* Products Management
* Product Order Monitoring
* Feedback Management
* Barber Schedule Management

## Technologies Used

* Android Studio
* Kotlin

* GCash Payment Integration
